什么是面向对象?

面向对象就是把数据和对数据的操作放在一起 作为一个相互联系的整天 对象  # 把他与现实生活联系起来 方便人的理解和程序的制作

例如接口 是实现共同的方法(行为) 包含过来做为一个整体

抽象类 把事物的基本特性提取出来作为一个类 

接口和抽象类是不能被实例化的 接口是用于存放静态的常量和其他一些方法 具体实现要通过重写完成



文件输入输出流的作用

文件输入数据可以用于文件的复制也用于 网络中数据的通信  用与程序的数据交互
一般都是利用字节流处理文件 通过字节流的复制和处理可以减少错误的发生
